<h1 align="center">Resource Groups, Storage Account, and Containers</h1>

This is a tutorial on how to create a resource group, storage account, and container on Microsoft Azure

<h2>Technology used</h2>
Microsoft Azure

<h2>Prequisite</h2>
Must have a subscription account on Azure




<h3 align="center"> 1. How to create a resource group</h3>

- On the homepage of <a href="www.portal.azure.com"> Azure</a>, search for resource group and click on the resource group icon.

<img src=https://github.com/Archie735/resource-group-storage-container/assets/150314129/45a39293-2dc4-42ae-9218-bc7f6e58a138 width="10%" height="auto" alt="resource group icon"/>


- Click "Create a resource group".
- Select your subscription then name the resource group.
- Choose the desired region

<h3 align="center"> 2. How to create a storage account</h3>

- On Azure's homepage, search Storage Accounts and click on the Storage Accounts icon.

<img src=https://github.com/Archie735/resource-group-storage-container/assets/150314129/c6130d13-7bb5-4638-88eb-da76f9157a4f width="12%" height="auto" alt="storage accounts icon"/>

- Click "Create storage account"
- Select your subscription and your desired resource group
- Name your account
- Make sure the region is the same as the resource region
- Click review
- Create
  
<img src=https://github.com/Archie735/resource-group-storage-container/assets/150314129/712e3a8f-10e9-4f1a-baef-ead7ac610d76 width="50%" height="auto" alt="storage account"/>
